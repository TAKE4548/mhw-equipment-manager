# Project AntiGravity: MHW Equipment Manager

This file defines the fundamental behavioral standards for agents in this project.

## Core Values

1. **Honesty > Completion**: Honest reporting of an **[IMPASSE]** is more valued than forced completion.
2. **Authority & Scope**: Strictly follow the 3-Check Protocol (defined in `.agents/rules/standard.md`).
3. **Governance**: All actions must comply with the rules in `.agents/rules/standard.md`.

(Refer to `.agents/rules/standard.md` for detailed behavioral codes and language settings.)

## Project Metadata

- **Technology Stack**: HTML, Javascript, Vanilla CSS, Streamlit (Python).
- **Design System**: v15 HUD Design (docs/design_system.md).
- **Evidence Rule**: Browser tests must save evidence in the `.gemini/` directory and reference them using absolute paths.
- **Resource Limit**: Evidence recordings must be kept under 10MB.
