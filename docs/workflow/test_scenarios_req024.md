# Manual Test Cases: REQ-024 武器選択UIの刷新

### MT-001: 階層フィルタの連動確認
- **Pre-condition**: 複数の武器種、属性、スキルを持つ武器が複数登録されている。
- **Steps**:
  1. 「復元強化厳選」ページを開く。
  2. 「武器種」を選択する。
  3. 「属性」の選択肢が、選択した武器種の武器が持つ属性のみに絞り込まれていることを確認する。
  4. 「スキル」の選択肢が、さらに絞り込まれた結果に基づいていることを確認する。
- **Expected Result**: 段階的に選択肢が絞り込まれ、「候補: N件」の数値が動的に更新される。

### MT-002: レスポンシブグリッドの表示確認
- **Pre-condition**: 候補となる武器が4件以上存在する。
- **Steps**:
  1. デスクトップ表示（全画面）で、武器カードが最大3カラムで並んでいることを確認する。
  2. ブラウザの幅を狭めていき、2カラム、1カラムへと動的に折り返されることを確認する。
- **Expected Result**: カードが画面幅をはみ出したり潰れたりせず、きれいに整列している。

### MT-003: 武器選択と登録フォームの表示
- **Pre-condition**: 武器が1件以上表示されている。
- **Steps**:
  1. 武器カードの「❯」ボタンをクリックする。
  2. クリックした武器がゴールド枠（選択状態）になることを確認する。
  3. グリッドの下に「2. 抽選結果を入力」セクションが表示されることを確認する。
- **Expected Result**: 選択した武器が即座に強調され、入力フォームが露出する。
