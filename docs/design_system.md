# Design System: MHWs Equipment Manager

## デザイン指針
視認性と操作性を両立するため、データ密度の高いコンパクトなビジュアルと識別しやすいカラーリングを標準とする。

## コンポーネントデザイン
- **超スリム・シングル行カード**: 
  - 1レコード1行。全画面で「ウルトラスリム」なカード形式を採用し、高密度での情報一覧性を確保する。
- **属性別カラーバッジ**: 
  - 属性名称のみ（火, 水, 雷...）を表示する単体バッジ。背景色はモンハン標準の属性イメージカラー（MHカラー）を適用する。

## アクセシビリティ・レイアウト
- 余白を極力減らし、固定幅カラムを多用して位置ズレを防ぐ。
- Undo/Redoなどの操作系ボタンはツールバーに並べて配置する。

## 状態遷移と比較のビジュアル標準
情報の「変化（Before/After）」を示す際は、以下の **Visual Band** 形式を適用する。
- [x] **鑑定護石（ランダム護石）管理機能 (Completed)**
  - [x] Rarity (レア度) ごとのスキル・スロットバリデーション機能
  - [x] スキルグループ・レア度パターン管理用の外部マスタデータ化 (`talisman_master.json`)
  - [x] **プログレッシブ・カスケード制御の導入**:
    - [x] レア度→スキル1→スキル2→スキル3→スロットの順で入力を強制し、未入力時は後段を無効化するUI。
    - [x] 「スキル選択」→「レベル選択」を同一ダイアログで完結させる2ステップUIの実装。
    - [x] **依存関係のリセット**: レア度や上流スキルが変更された際、それ以降の下流データを自動クリアする挙動。
  - [x] スリムデザインを採用した護石一覧・一括管理画面の実装

- [ ] **強化厳選画面のリファクタリング (Planned)**
  - [ ] **Visual Band 遷移 UI**: 比較表の「現/目」ラベルを廃止し、左端のカラーバーと背景輝度、および5スロット完全整列のグリッドによる直感的な推移表現。
  - [ ] **武器選択カードの情報拡充**: シリーズスキルに加え、グループスキルと生産ボーナス（3枠）をカード内に表示。
  - [ ] **厳選リストのフィルタ・ソート**: 登録済みトラッキング一覧に対し、武器種・属性・残り回数等での絞り込みと並び替え機能の実装。長さによるレイアウト崩れを許容しない。

- **Before（現在）**: 
    - 左端に `2px` のグレー (`#666`) バーを表示。
    - `font-size: 0.85rem`、不透明度 `0.6` 程度で muted に表示。背景は極めて薄い透過。
- **After（目標）**: 
    - 左端に `4px` の鮮やかな緑（`#2ecc71` などの成功カラー）バーを表示。
    - **`font-size: 1.2rem`** 程度まで拡大し、`font-weight: bold` で目標状態を強調。
    - 変化（強化）のあった箇所はさらに輝度を上げ、セット全体としての視認性を「攻めた」比率に調整。
- **配置**: 上下に並べる際、目標（After）側を視覚的重心とし、これまでのスリムデザインを崩さない範囲でリスト内での存在感を高める。
- **整列要件**: データのズレを防ぐため、常に固定幅グリッド（例：5カラム等幅）内で表示し、文字列の長さによるレイアウト崩れを許容しない。
